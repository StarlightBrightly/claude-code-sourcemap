# Claude Code 架构概览

## 项目基本信息

- **项目名称**: @anthropic-ai/claude-code
- **版本**: 2.1.88
- **类型**: CLI 工具 + TUI (Terminal User Interface)
- **运行时**: Bun (支持 Node.js)
- **UI 框架**: React + Ink (Terminal React 渲染器)
- **语言**: TypeScript

## 核心技术栈

### 1. 运行时与构建
- **Bun**: 主要运行时,提供快速的启动和执行
- **Webpack**: 打包工具,生成 sourcemap
- **TypeScript**: 类型系统

### 2. UI 层
- **React**: 组件化 UI
- **Ink**: 将 React 组件渲染到终端
- **Yoga Layout**: Flexbox 布局引擎 (用于终端布局)
- **Chalk**: 终端颜色

### 3. CLI 框架
- **Commander.js**: 命令行参数解析
- **Inquirer**: 交互式命令行提示

### 4. 核心功能库
- **@anthropic-ai/sdk**: Claude API 客户端
- **@modelcontextprotocol/sdk**: MCP (Model Context Protocol) 支持
- **vscode-jsonrpc**: LSP (Language Server Protocol) 通信
- **source-map**: Sourcemap 处理

## 启动流程

### 1. 入口点 (main.tsx)

```typescript
// 性能优化: 并行预加载
profileCheckpoint('main_tsx_entry')
startMdmRawRead()              // MDM 配置读取
startKeychainPrefetch()        // Keychain 预取

// 初始化
init()                         // 基础初始化
initializeGrowthBook()         // 特性开关
fetchBootstrapData()           // 引导数据

// 启动 REPL
launchRepl()
```

### 2. 关键优化

**并行预加载策略**:
- MDM 配置读取 (~65ms)
- Keychain 读取 (OAuth + API key)
- 在模块加载期间并行执行,节省启动时间

### 3. 特性开关 (Feature Flags)

使用 `feature()` 函数进行条件编译:

```typescript
const coordinatorModeModule = feature('COORDINATOR_MODE')
  ? require('./coordinator/coordinatorMode.js')
  : null

const assistantModule = feature('KAIROS')
  ? require('./assistant/index.js')
  : null
```

支持的特性:
- `COORDINATOR_MODE`: 多 Agent 协调模式
- `KAIROS`: 助手模式
- `BRIDGE_MODE`: 桥接模式
- `VOICE_MODE`: 语音模式
- `AGENT_TRIGGERS`: Agent 触发器
- `PROACTIVE`: 主动模式

## 核心设计模式

### 1. 工具系统 (Tool System)

所有工具继承自 `Tool` 基类:

```typescript
interface Tool {
  name: string
  description: string
  inputSchema: ToolInputJSONSchema
  execute(params: any, context: ToolContext): Promise<ToolResult>
}
```

### 2. 命令系统 (Command System)

命令通过 Commander.js 注册:

```typescript
interface Command {
  name: string
  description: string
  action: (options: any) => Promise<void>
}
```

### 3. 状态管理

使用自定义 Store 模式:

```typescript
const store = createStore<AppState>(getDefaultAppState())
store.subscribe(onChangeAppState)
```

### 4. 权限系统

三种权限模式:
- **Auto**: 自动批准常见操作
- **Supervised**: 需要用户确认
- **Manual**: 完全手动控制
