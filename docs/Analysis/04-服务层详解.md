# Claude Code 服务层详解

## 服务层架构

服务层提供核心业务逻辑和外部集成功能。

## 核心服务

### 1. API 服务 (services/api/)

**Claude API 集成**:
- `bootstrap.ts`: 引导数据获取
- `filesApi.ts`: 文件上传/下载
- `dumpPrompts.ts`: 提示符转储 (调试)
- `promptCacheBreakDetection.ts`: 缓存失效检测

**关键功能**:
```typescript
// 获取引导数据
fetchBootstrapData(): Promise<BootstrapData>

// 上传会话文件
uploadSessionFiles(files: File[]): Promise<void>
```

### 2. MCP 服务 (services/mcp/)

**Model Context Protocol 集成**:

#### 2.1 客户端管理 (client.ts)

```typescript
// 连接 MCP 服务器
connectToServer(config: McpServerConfig): Promise<MCPServerConnection>

// 获取工具列表
fetchToolsForClient(client: Client): Promise<Tools>

// 获取资源列表
fetchResourcesForClient(client: Client): Promise<Resources>
```

#### 2.2 传输层支持

- **StdioTransport**: 标准输入输出
- **SSETransport**: Server-Sent Events
- **WebSocketTransport**: WebSocket
- **StreamableHTTPTransport**: HTTP 流式传输

#### 2.3 配置管理 (config.ts)

```typescript
// 解析 MCP 配置
parseMcpConfig(path: string): McpServerConfig[]

// 获取 MCP 服务器签名
getMcpServerSignature(config: McpServerConfig): string
```

#### 2.4 官方注册表 (officialRegistry.ts)

预定义的官方 MCP 服务器:
- `filesystem`: 文件系统访问
- `github`: GitHub 集成
- `gitlab`: GitLab 集成
- `google-drive`: Google Drive
- `slack`: Slack 集成

### 3. LSP 服务 (services/lsp/)

**Language Server Protocol 集成**:

```typescript
// 初始化 LSP 服务器
initializeLspServerManager(): Promise<void>

// 获取诊断信息
getDiagnostics(uri: string): Promise<Diagnostic[]>

// 代码补全
getCompletions(uri: string, position: Position): Promise<CompletionItem[]>
```

### 4. 分析服务 (services/analytics/)

#### 4.1 GrowthBook (growthbook.ts)

特性开关和 A/B 测试:

```typescript
// 初始化 GrowthBook
initializeGrowthBook(): Promise<void>

// 获取特性值
getFeatureValue(key: string, defaultValue: any): any
```

#### 4.2 事件追踪 (index.ts)

```typescript
// 记录事件
logEvent(event: string, metadata: AnalyticsMetadata): void
```

### 5. 上下文压缩服务 (services/compact/)

**自动压缩长对话**:

#### 5.1 压缩策略 (compact.ts)

```typescript
// 执行压缩
compactMessages(messages: Message[]): Promise<Message[]>

// 微压缩 (增量)
microCompact(messages: Message[]): Promise<Message[]>
```

#### 5.2 分组策略 (grouping.ts)

将相关消息分组以提高压缩效率:
- 工具调用序列
- 文件编辑序列
- 搜索操作序列

#### 5.3 自动压缩 (autoCompact.ts)

```typescript
const COMPACT_THRESHOLD = 0.85  // 85% 上下文使用率触发压缩
```

### 6. 插件服务 (services/plugins/)

**插件管理**:

```typescript
// 加载插件
loadPlugin(path: string): Promise<Plugin>

// 初始化插件
initializePlugin(plugin: Plugin): Promise<void>

// 卸载插件
unloadPlugin(pluginId: string): Promise<void>
```

### 7. 会话内存服务 (services/SessionMemory/)

**持久化会话状态**:

```typescript
// 保存会话内存
saveSessionMemory(sessionId: string, data: any): Promise<void>

// 加载会话内存
loadSessionMemory(sessionId: string): Promise<any>
```

### 8. 远程管理设置 (services/remoteManagedSettings/)

**企业配置管理**:

```typescript
// 加载远程设置
loadRemoteManagedSettings(): Promise<Settings>

// 刷新远程设置
refreshRemoteManagedSettings(): Promise<void>
```

### 9. 策略限制服务 (services/policyLimits/)

**企业策略执行**:

```typescript
// 检查策略
isPolicyAllowed(action: string): boolean

// 加载策略限制
loadPolicyLimits(): Promise<PolicyLimits>
```

## 服务间通信

### 1. 事件总线

服务通过事件总线通信:

```typescript
eventBus.emit('mcp:connected', { server: 'github' })
eventBus.on('session:ended', handleSessionEnd)
```

### 2. 依赖注入

服务通过上下文传递:

```typescript
interface ServiceContext {
  api: ApiService
  mcp: McpService
  lsp: LspService
  analytics: AnalyticsService
}
```
