# Claude Code 状态管理与数据流

## 状态管理架构

Claude Code 使用自定义的状态管理系统,基于 Store 模式。

## AppState 结构

### 1. 核心状态

```typescript
interface AppState {
  // 配置
  settings: SettingsJson
  mainLoopModel: ModelSetting

  // UI 状态
  statusLineText: string | undefined
  expandedView: 'none' | 'tasks' | 'teammates'

  // 消息与历史
  messages: Message[]
  conversationHistory: Message[]

  // 工具与权限
  tools: Tool[]
  toolPermissionContext: ToolPermissionContext
  permissionMode: PermissionMode

  // MCP 连接
  mcpClients: MCPServerConnection[]
  mcpResources: ServerResource[]

  // 任务管理
  tasks: TaskState[]

  // Agent 状态
  agents: AgentDefinitionsResult
  activeAgents: Map<AgentId, AgentState>

  // 插件
  plugins: LoadedPlugin[]

  // 通知
  notifications: Notification[]
}
```

### 2. 推测执行状态

```typescript
type SpeculationState =
  | { status: 'idle' }
  | {
      status: 'active'
      id: string
      abort: () => void
      messages: Message[]
      boundary: CompletionBoundary | null
    }
```

## Store 实现

### 1. 创建 Store

```typescript
const store = createStore<AppState>(getDefaultAppState())
```

### 2. 订阅变化

```typescript
store.subscribe((newState, oldState) => {
  // 响应状态变化
  onChangeAppState(newState, oldState)
})
```

### 3. 更新状态

```typescript
store.setState({
  ...store.getState(),
  messages: [...messages, newMessage]
})
```

## 数据流

### 1. 用户输入流

```
用户输入 → 处理 → 创建 UserMessage → 添加到 messages → 触发 query
```

### 2. AI 响应流

```
API 请求 → 流式响应 → 解析 → 创建 AssistantMessage → 更新 messages
```

### 3. 工具调用流

```
工具请求 → 权限检查 → 执行 → 结果 → 创建 ToolResultMessage → 继续对话
```

## 消息类型

### 1. UserMessage

```typescript
interface UserMessage {
  role: 'user'
  content: string | ContentBlock[]
  timestamp: number
}
```

### 2. AssistantMessage

```typescript
interface AssistantMessage {
  role: 'assistant'
  content: string | ContentBlock[]
  toolUses?: ToolUseBlock[]
  timestamp: number
}
```

### 3. SystemMessage

```typescript
interface SystemMessage {
  role: 'system'
  content: string
  timestamp: number
}
```

### 4. ToolResultMessage

```typescript
interface ToolResultMessage {
  role: 'user'
  content: ToolResultBlock[]
  timestamp: number
}
```

## 状态持久化

### 1. 会话存储

```typescript
// 保存会话
saveSession(sessionId: string, messages: Message[]): void

// 加载会话
loadSession(sessionId: string): Message[]
```

### 2. 配置存储

```typescript
// 保存配置
saveGlobalConfig(config: Config): void

// 加载配置
getGlobalConfig(): Config
```

通过 React Context 共享状态:

```typescript
<AppStateContext.Provider value={store}>

```

### 2. 跨进程同步

通过文件系统同步状态:
