# Claude Code UI 渲染机制

## UI 架构

Claude Code 使用 React + Ink 在终端中渲染 UI。

## Ink 框架

### 1. 核心概念

Ink 将 React 组件渲染到终端:

```typescript
import { render } from 'ink'
import React from 'react'

render(<App />)
```

### 2. 布局引擎

使用 Yoga Layout (Flexbox):

```typescript
<Box flexDirection="column" padding={1}>
  <Text>Header</Text>
  <Box flex={1}>Content</Box>
</Box>
```

## 核心组件

### 1. REPL 组件

主交互界面:

```typescript
<REPL
  messages={messages}
  onSubmit={handleSubmit}
  tools={tools}
/>
```

### 2. 消息渲染

```typescript
<MessageList>
  {messages.map(msg => (
    <Message key={msg.id} message={msg} />
  ))}
</MessageList>
```

### 3. 工具调用显示

```typescript
<ToolUse
  toolName={toolName}
  status={status}
  progress={progress}
/>
```

### 4. 权限提示

```typescript
<PermissionPrompt
  tool={tool}
  onApprove={handleApprove}
  onDeny={handleDeny}
/>
```

## 终端 I/O

### 1. 输入处理

```typescript
// 键盘事件
useInput((input, key) => {
  if (key.return) {
    handleSubmit()
  }
})
```

### 2. 输出优化

- 增量渲染
- 虚拟滚动
- 输出截断

### 3. ANSI 转义序列

```typescript
// 颜色
chalk.blue('text')

// 光标控制
process.stdout.write('\x1b[2J')  // 清屏
```

## 性能优化

### 1. 渲染优化

- React.memo 缓存组件
- 虚拟化长列表
- 延迟渲染

### 2. 输出限制

```typescript
const MAX_OUTPUT_LINES = 1000
const TRUNCATE_THRESHOLD = 10000
```
