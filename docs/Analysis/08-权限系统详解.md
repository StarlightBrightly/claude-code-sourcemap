# Claude Code 权限系统详解

## 权限模式

### 1. Auto 模式

自动批准常见操作:
- 文件读取
- 代码搜索
- 安全的 Shell 命令

### 2. Supervised 模式

需要用户确认:
- 文件写入
- 文件删除
- 危险命令

### 3. Manual 模式

所有操作都需要确认。

## 权限规则

### 1. 工具权限

```typescript
interface ToolPermission {
  tool: string
  allowed: boolean
  patterns?: string[]
}
```

### 2. 路径权限

```typescript
interface PathPermission {
  path: string
  read: boolean
  write: boolean
}
```

### 3. 命令权限

```typescript
interface CommandPermission {
  command: string
  allowed: boolean
}
```

## 权限检查流程

```
工具调用 → 检查权限规则 → 匹配模式 → 返回结果
```

## 权限持久化

权限决策可以保存:
- 会话级别
- 全局级别
- 项目级别
