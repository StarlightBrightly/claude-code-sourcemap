# Claude Code 技能系统详解

## 技能概述

技能 (Skills) 是可复用的提示符模板,可以快速调用预定义的任务。

## 技能结构

### 1. 技能定义

```markdown
---
name: commit
description: Create a git commit
---

Follow these steps to create a commit:
1. Run git status
2. Review changes
3. Create commit message
4. Execute commit
```

### 2. 参数化技能

```markdown
---
name: review-file
description: Review a specific file
args:
  - name: file
    description: File path to review
---

Review the file at {{file}} and provide feedback.
```

## 技能调用

### 1. 通过命令调用

```bash
/commit
/review-file src/app.ts
```

### 2. 通过工具调用

```typescript
SkillTool({
  skill: 'commit',
  args: ''
})
```

## 内置技能

### 1. Git 相关

- `commit`: 创建提交
- `review`: 代码审查
- `commit-push-pr`: 提交并创建 PR

### 2. 代码相关

- `simplify`: 简化代码
- `loop`: 循环执行任务

### 3. 配置相关

- `update-config`: 更新配置
- `keybindings-help`: 快捷键帮助

## 技能目录

- 内置技能: `src/skills/bundled/`
- 用户技能: `~/.claude/skills/`
- 项目技能: `.claude/skills/`

## 技能组合

技能可以组合使用:

```markdown
---
name: full-review
---

First, run the /commit skill.
Then, run the /review skill.
Finally, create a PR.
```
