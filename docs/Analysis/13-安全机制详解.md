# Claude Code 安全机制详解

## 安全架构

Claude Code 实现了多层安全机制保护用户系统。

## 命令安全

### 1. Bash 命令解析

使用 AST 解析检测危险命令:

```typescript
parseForSecurity(command: string): SecurityResult
```

### 2. 危险命令检测

- `rm -rf /`
- `:(){ :|:& };:`  (Fork bomb)
- 未授权的网络访问
- 系统文件修改

### 3. 沙箱模式

```typescript
shouldUseSandbox(command: string): boolean
```

## 文件系统安全

### 1. 路径验证

```typescript
// 防止路径遍历
safeResolvePath(path: string): string
```

### 2. 只读约束

```typescript
checkReadOnlyConstraints(path: string): boolean
```

### 3. 文件权限检查

验证文件访问权限。

## 网络安全

### 1. HTTPS 强制

HTTP 自动升级到 HTTPS。

### 2. 代理支持

```typescript
getProxyFetchOptions()
getWebSocketProxyAgent()
```

### 3. mTLS 支持

```typescript
getWebSocketTLSOptions()
```

## 数据安全

### 1. 敏感信息过滤

自动过滤:
- API 密钥
- 密码
- Token

### 2. 安全存储

```typescript
// macOS Keychain
// Windows Credential Manager
// Linux Secret Service
```

### 3. 日志脱敏

日志中自动移除敏感信息。

## 企业安全

### 1. MDM 集成

支持 MDM (Mobile Device Management) 配置。

### 2. 策略执行

```typescript
isPolicyAllowed(action: string): boolean
```

### 3. 审计日志

记录所有关键操作。
