# Claude Code 关键技术点总结

## 1. 架构设计亮点

### 模块化设计
- 工具系统高度可扩展
- 插件系统支持第三方扩展
- MCP 协议实现标准化集成

### 性能优化
- 启动时并行预加载 (节省 ~65ms)
- 条件编译实现死代码消除
- 流式处理减少延迟

### 用户体验
- 三种权限模式平衡安全与便利
- 自动后台化长时间任务
- 智能上下文压缩

## 2. 核心技术栈

### 运行时
- **Bun**: 快速启动和执行
- **Node.js**: 兼容性支持

### UI 层
- **React**: 组件化
- **Ink**: 终端渲染
- **Yoga**: Flexbox 布局

### 协议支持
- **MCP**: 外部工具集成
- **LSP**: 语言服务器
- **JSON-RPC**: 通信协议
