# 文档索引

## 文件说明

- `project-analysis.md`：面向整体理解的完整项目分析，覆盖项目定位、运行链路、核心子系统、技术栈判断、工程特点与局限。

## 阅读建议

如果你想先理解“这是什么项目、怎么跑起来、主要由哪些系统组成”，优先阅读 `project-analysis.md`。

如果你想快速查“某个目录里有什么、命令和工具有哪些、哪里最复杂”，优先阅读 `module-inventory.md`。
